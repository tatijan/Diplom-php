Закачать архив сервиса в корневую папку вашего сайта и распаковать. Создать базу данных для сайта. В файле .env скорректировать блок по базе данных: название базы данных, логин, пароль. Выполнить импорт базы данных из файла faq.sql Произвести настройку доступа к папке public согласно инструкций вашего хостинга для сайта на фреймворке Laravel.
